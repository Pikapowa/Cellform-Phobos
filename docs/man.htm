1.1 - Vérification avant installation

Dans cette première partie nous allons voir comment configurer votre serveur web pour que Cellform puisse fonctionner.

Tout d'abord vérifier que vous utilisez l'un de ces serveurs web :

  • Apache
  • Nginx

En effet Cellform fonctionne par defaut uniquement avec ces deux serveurs web, ceci est dû à la nécéssité d'activer la réécriture des urls, si vous voulez adapter la plateforme à un serveur non listé il vous faudra convertir le fichier .htaccess dans le format voulu.

Dans un second temps vérifiez que votre version de PHP soit supérieure à la 5.4.0 avec l'option magic_quotes désactivée.

1.2 - Lancer l'installation

Cellform à été conçu pour être le plus simple d'utilisation possible, y compris dans son module d'installation, cependant si vous rencontrez un problème le manuel ci-présent vous accompagne pendant tout le processus.

Pour commencer, vous devez télécharger l'archive tar.gz à cette adresse : cellform.tar.gz puis extraire la tarball dans votre racine web, classiquement sur linux celle-ci ce trouve dans /var/www/, sur windows dans c:/wamp/www.

Une fois l'opération terminée ouvrez votre navigateur web puis accèdez au module d'installation à cette adresse : http://localhost/install/index.php, attention à ne pas omettre "index.php" à la fin de l'url (n'oubliez pas que le mod_rewrite est activé).

Vous devez voir une page d'introduction des plus banale, suivie de la licence GPL dans sa version 3.0.

Bien, à partir d'ici les choses sérieuses commencent, c'est la page des pré-requis, vous avez toutes les explications dans la section suivante [1.3].

1.3 - Vérification des extensions et modules

Liste des extensions PHP vitales :

  • GD Image - Nécéssaire pour le traitement des images
  • JSON - Nécéssaire pour la liaison avec l'api Javascript
  • SimpleXML - Nécéssaire pour la gestion des traductions

Liste des extensions PHP non vitales :

  • CURL - Nécéssaire uniquement si la connexion via OAuth est activée

Liste des modules HTTP obligatoires :

  • Mod_rewrite - Nécéssaire pour le bon fonctionnement du routeur.
  • MySQL - Nécéssaire pour la sauvegarde des données (version supérieur ou égale à 4.1.0)

Si un module ou une extension requise n'est pas installée, le bouton [suivant] restera grisé.

1.3 - Vérification des droits sur les fichiers

La vérification des droits sur les fichiers fait partie du processus obligatoire lors de l'installation.

Ainsi tous les fichiers / dossiers suivants doivent être en mode libre écriture (et de préférence appartenir au groupe de votre serveur web) :

  • /cache/
  • /config/config.conf.php
  • /media/img/
  • /media/img/min/
  • /media/avatars/
  • /images/loginbox/bg.jpg
  • /images/loginbox/present.jpg
  • /images/media/banner.png

Une fois ceci fait, le bouton [suivant] devient bleu, vous pouvez donc continuer votre installation.

2.1 - Configuration générale

Dans cette section rien de vraiment compliqué, commencez par entrer le nom de votre site (je laisse libre court à votre imagination), vient ensuite le titre, une brève description qui sera affichée en page de connexion, l'adresse mail reservée a votre site (voir votre hébergeur), la langue par defaut à utiliser et enfin la clé de sécurité qui doit être un ensemble de chiffres et de lettres, celle-ci sera utilisée pour renforcer le système de cryptage de la plateforme.

2.2 - Configuration de la base de donnée

Cellform est pour l'instant uniquement compatible avec MySQL (cependant son code est prévu pour accueillir le support des autres SGBD).

Vous devez donc entrer vos identifiants de connexion à votre serveur SQL ainsi que le nom de la base reservé à la plateforme.

Dans le cas ou vos identifiants sont invalides, la plateforme vous signalera qu'elle ne peut pas se connecter au serveur SQL.

2.3 - Installation des tables

C'est à ce moment que le module d'installation injecte toutes les tables dans la base de donnée, si une erreur survient reportez vous à vos logs SQL.

Je précise que l'injection est effective uniquement si les tables ne sont pas inscrites dans la base.

2.4 - Configuration de l'administration

Vous pouvez maintenant vous inscrire sur le site avec les droits Administrateur (notez que cellform est conçu pour avoir un seul admin).

Une fois ceci fait vous pouvez enfin vous connecter !
